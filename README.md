# API 接口文档
web版简易在线API接口文档


## Description
* 入口是根目录的index.html
* api.js是公共的说明文件，及加载的分模块js，可以自定义修改
* apis目录是对应的每个模块的接口说明，每个模块一个独立的js文件。
